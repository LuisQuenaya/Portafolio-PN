Tarea 4: Método de Bisección ▶

Abrir PDF
Enunciado: Implementar el método de bisección para la función f(x) = x³ - x - 2 y aproximar la raíz entre a=1 y b=2. 
def f(x):
    return x**3 - x - 2

a, b = 1, 2
for _ in range(10):
    c = (a + b)/2
    if f(a)*f(c) < 0:
        b = c
    else:
        a = c
print("Raíz aproximada:", c)

def f(x): return x**2 - 2
def df(x): return 2*x

x = 1
for i in range(5):
    x = x - f(x)/df(x)
print("Raíz aproximada:", x)
